About Me

I also have experience in Application development in languages such as Python, Visual Basic, C and C#. I have been working with programming languages since 2017, I started with Python, HTML and CSS. RPG Dialogue System


For my Final year project at University I tasked myself to create an RPG style Dialogue and Quest System



When designing the project my aim was to create a dialogue system similar to what you may find in RPGs, my main influence for this was The Elder Scrolls series. I wanted to create a system where the player could enter dialogue with any character, the player should be able to respond to questions and ask questions in return.

Along side this, I wanted to build in a basic quest system that would help guide the player to key characters. I wanted each dialogue to feel somewhat unique and relatively dynamic, so the player feels a character in the world and not that the player is the world.

Escape The Stu-dio


Trapped in the Stu-dio late at night you must solve increasing more complex puzzles to escape. Fix wires, crawl through vents, solve dangerous puzzles and escape. But remember. You are not alone.



Escape The Stu-dio is a collaborative game developed by junior and senior students as part of university collaboration module within the Unreal 5 Engine. The game is a first person puzzle game with horror elements developed with a six-week timescale. I was a senior in the lead tech role, most of my tasks in the pre production stage were talking with the designers to determin what was possible with the enforced timescale.

In producion I was responsible for delegating tasks to my tech team and helping to keep GitHub running smoothly, other responsiblites was helping juniors. Late into production I was responsible for importing assets and animations in engine, major bugfixing and Publication.

Mote Valley


When an unyielding darkness begins encroaching on the world, you turn to nurturing the mysterious Motes to release light and fend it off. What will you do to rid the world of darkness?



Mote Valley is a collaborative game developed by junior and senior students as part of university collaboration module within the Unreal 4 Engine. The game is a first person exploration farming game developed with a six-week timescale. I was a junior in the tech role, within the tech department we didn't have a reporting senior and had to rely on our own abilities with the occasional direction of the project lead.

I was responsible for some of the games key mechanics mainly being the darkness system and the timer that made it shrink toward the spawn and the the light system that would expand the darkness sphere when a mote was leveled up or released.

My Contributions
Developed the darkness sphere
Developed player health and respawns
Created timers for the darkness sphere
Developed the timer for how long the player can be outside the light
Added the ability to grow the sphere based on the mote light level
Created darkness HUD elements

Arima Railways


Arima Railways is a ticketing application developed as part of a college final year project.

The user is able to select how many tickets they wish to buy from 3 age catagories and the price is displayed in real time. This project was developed using Visual Basic
