About Me

I am a Web and Games Developer with experience in Unreal Engine and Unity as well as HTML, CSS, JavaScript, EJS, Node.js and more. I also have experience in Application development in languages such as Python, Visual Basic, C and C#. I have been working with programming languages since 2017, I started with Python, HTML and CSS. Following this I worked with JavaScript and Bootstrap to allow for more robust websites. I have since worked on Windows Applications in Visula Basic, C and C#. Since 2020 I have been learning Unreal Engine and Unity throughout University developing games on the Design and Tech Development side. I have Completed Game Jams as well as Collaborative games development alongside individual projects. I am currently studying BSc (Hons) Computer Gameplay Design and Production in my final year.
